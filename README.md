# Robotic Arm Project

## Project Description
This robotic arm uses gears, belts, and motors to move. It is 0.75× scale. The design focuses on strength and flexibility. Using motors and mechanical linkages instead of servos allows more control over movement and load. The project combines mechanical design, electronics, and coding into one working system. It shows how parts, wiring, and software work together.

## Why I Made This Project
I built this arm to improve my skills in design, tolerances, electronics, and coding. Controlling motors with gears and belts requires careful planning and testing. I wanted to learn how mechanical systems behave under load and how to control them precisely. This project gave me hands-on experience with real-world engineering challenges while keeping the design simple and understandable.
